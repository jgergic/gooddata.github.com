GoodData CL

A quick, easy-to-use command-line utility that wraps access to our APIs. It executes short scripts that can be run and configured automatically. Features include:

  • create projects — allows you to automatically create new projects on the fly
  • create a data model — helps you describe relationship in your data and create a logical data model in GoodData
  • extract data from DBs, online apps — built-in connectors to any JDBC driver, Google Analytics, Salesforce
  • …and even more — partial loads, data snapshotting, time dimension mapping and more…
By being command-line driven, it can be easily included as part of an automated integration workflow.
